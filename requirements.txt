# This project now uses uv for reproducible dependency management.
#
# Setup on any machine:
#   uv sync
#
# Add a dependency:
#   uv add <package>
#
# Update lockfile after manual pyproject.toml edits:
#   uv lock
#
# See pyproject.toml for top-level deps and uv.lock for the full lockfile.
#
# Notes:
# - Keep separate envs for `deepspeed`, `sglang`, and `nemo`.
# - `LLM-baseline` stays on its own `uv` workflow.
